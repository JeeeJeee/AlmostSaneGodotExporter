# Almost sane godot exporter

I have found Godots 3D workflow to be extremely painful to work with, so I spent some time to make this.  
The project is based on the [blender-for-unrealengine](https://github.com/xavier150/Blender-For-UnrealEngine-Addons/). But with some parts rewritten and a lot of parts stripped out.

### Features
*This addon is focused on meshes. No effort was made to support animations/actions etc.*
- Exporting objects as Box/Convex collisions
- Setting collision layers and masks that Godot will use
- Convenience functions for scene export (*moving/rotating objects to zero*)
